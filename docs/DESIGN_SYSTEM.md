# Design System

**Project:** CAMARA Network API Experience Centre

**Version:** 1.0

---

# Purpose

The Design System provides a consistent visual language for the Experience Centre.

Every page, component and interaction should feel like part of the same product.

---

# Design Philosophy

Inspired by:

- Apple
- Stripe
- Vercel
- Linear

Characteristics:

- Clean
- Modern
- Professional
- Technical
- Trustworthy
- Minimal

---

# Brand Personality

The Experience Centre should feel:

- Intelligent
- Confident
- Innovative
- Approachable
- Premium

Never corporate.

Never cluttered.

---

# Layout

Maximum content width:

1280px

Spacing:

8px spacing scale

Generous whitespace.

Never overcrowd pages.

# Status Badges

Available

Green

In Progress

Amber

Planned

Slate

Deprecated

Red

---

# Icons

Lucide Icons

Use icons to reinforce understanding.

Never decorate unnecessarily.

---

# Motion

Animations should:

- Explain
- Guide
- Reinforce

Avoid decorative animation.

---

# Cards

Cards should:

- Have generous spacing
- Rounded corners
- Soft shadows
- Clear hierarchy

---

# Buttons

Primary

Filled

Secondary

Outlined

Ghost

Minimal

Danger

Destructive actions only

---

# Responsive Design

Support:

Desktop

Tablet

Mobile

Mobile-first.

---

# Accessibility

WCAG AA minimum.

Keyboard navigation.

Visible focus states.

Semantic HTML.

---

# Dark Mode

Dark mode is a first-class experience.

Do not simply invert colours.

Design specifically for dark surfaces.

---

# Design Motto

Elegant.

Purposeful.

Consistent.

Every screen should inspire confidence.
